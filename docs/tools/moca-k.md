# MoCA-K (몬트리올 인지평가 한국판)

## 1. 기본 정보

**정식 명칭**: Montreal Cognitive Assessment - Korean Version (MoCA-K)
**목적**: 경도인지장애(MCI) 및 초기 치매 선별을 위한 간단한 인지기능 평가 도구
**소요 시간**: 약 10-15분
**대상**: 만 18세 이상 성인
**총점**: 30점 만점
**선별 기준점**: 22/23점 (22점 이하 인지장애 의심)

## 2. 원저자 및 출처

**원저자**: Ziad Nasreddine, MD
**원도구 개발**: 1996년 (캐나다 몬트리올)
**원논문**: Nasreddine, Z. S., et al. (2005). "The Montreal Cognitive Assessment, MoCA: A Brief Screening Tool For Mild Cognitive Impairment." *Journal of the American Geriatrics Society*, 53(4), 695-699.

**한국어판 표준화**: 강연욱, 박재설, 등 (2009)

**표준화 연구진**:
- 강연욱 (한림대학교 심리학과)

- 나덕렬 (성균관대학교 삼성서울병원 신경과)

**규준 자료**:

- 교육 수준별 규준 제공

## 5. 평가 영역 및 구조

MoCA-K는 8개 인지 영역을 평가하며, 총 12개 과제로 구성됩니다.

### 5.1 시공간/실행기능 (Visuospatial/Executive) - 5점
1. **교대연결선로검사** (Alternating Trail Making) - 1점
   - 숫자와 글자를 교대로 연결 (1-가-2-나-3-다-4-라-5)

2. **입방체 모사** (Cube Copy) - 1점
   - 3차원 입방체 그림 정확히 따라 그리기

3. **시계그리기** (Clock Drawing) - 3점
   - 윤곽(1점): 완전한 원 그리기
   - 숫자(1점): 12개 숫자 정확한 위치에 배치
   - 바늘(1점): 11시 10분 표시

### 5.2 이름대기 (Naming) - 3점
- 동물 그림 3개(사자, 코뿔소, 낙타) 이름대기
- 각 1점씩

### 5.3 주의력 (Attention) - 6점
1. **숫자 바로 따라하기** (Forward Digit Span) - 1점
   - 5개 숫자 정확히 반복

2. **숫자 거꾸로 따라하기** (Backward Digit Span) - 1점
   - 3개 숫자 역순으로 반복

3. **주의집중** (Vigilance) - 1점
   - 글자 읽기 과제에서 특정 글자('가') 들을 때마다 손 두드리기

4. **계산** (Serial 7s) - 3점
   - 100에서 7씩 연속 빼기 (93, 86, 79, 72, 65)
   - 각 정답 1점 (최대 3점)

### 5.4 언어 (Language) - 3점
1. **문장 따라하기** (Sentence Repetition) - 2점
   - 두 문장 정확히 따라하기 (각 1점)
   - 예: "나는 내가 언제나 다른 사람으로부터 도움을 받을 수 있다는 것을 알고 있다"

2. **유창성** (Fluency) - 1점
   - 1분 내 '기' 또는 '시'로 시작하는 단어 11개 이상 산출

### 5.5 추상적 사고 (Abstraction) - 2점
- 단어 쌍의 유사성 찾기 (각 1점)
- 예: "기차-자전거" → "탈 것" 또는 "교통수단"

### 5.6 지연회상 (Delayed Recall) - 5점
- 검사 초반에 학습한 5개 단어 회상
- 단어 목록 (예): 얼굴, 벨벳, 교회, 데이지, 빨강
- 각 단어당 1점 (단서 없이 자발적 회상만)

### 5.7 지남력 (Orientation) - 6점
- 날짜 (1점)
- 월 (1점)
- 연도 (1점)
- 요일 (1점)
- 장소 (1점)
- 도시 (1점)

### 5.8 교육 연수 보정
- **추가 점수 없음** (원점수 그대로 사용)
- 단, 해석 시 교육 연수 고려:
  - 6년 이하: +1점 가산
  - 7년 이상: 보정 없음

## 6. 채점 방식

### 6.1 일반 원칙
- 각 문항은 정확히 수행한 경우만 점수 부여
- 부분 점수 없음 (all-or-nothing)
- 검사자가 즉시 채점하며 피검자에게 점수 알리지 않음

### 6.2 세부 채점 기준

**시공간/실행기능**
- 교대연결선로: 오류 없이 완수 시 1점
- 입방체: 3차원 투시도법 정확, 모든 선 포함 시 1점
- 시계: 각 요소별 정확도 평가

**이름대기**
- 자발적으로 정확한 이름 말하면 1점
- 단서 제공하거나 선다형으로 물으면 0점

**주의력**
- 숫자 따라하기: 한 번의 시행에서 정확해야 함
- 계산: 자기교정 허용 (최종 답이 정확하면 점수 부여)

**언어**
- 문장 따라하기: 정확한 반복만 인정 (단어 추가/삭제/대체 불가)
- 유창성: 변형어, 고유명사, 반복 제외

**지연회상**
- 자발적 회상만 점수 (단서 회상이나 재인은 참고용)

**지남력**
- 정확한 정보만 점수 부여

### 6.3 총점 계산
```
총점 = 시공간/실행(5) + 이름대기(3) + 주의력(6) + 언어(3) +
       추상(2) + 지연회상(5) + 지남력(6)
     = 30점 만점

보정 점수 = 총점 + 교육보정(0 or 1)
```

## 7. 해석 기준

### 7.1 절단점 (Cut-off Scores)

**한국 규준 (강연욱 등, 2009)**
- **정상**: 23점 이상
- **경도인지장애(MCI) 의심**: 22점 이하
- **치매 의심**: 일반적으로 20점 이하

### 7.2 교육 수준별 고려사항

| 교육 연수 | 절단점 | 보정 |
|---------|--------|------|
| 6년 이하 | 21/22 | +1점 가산 |
| 7-12년 | 22/23 | 보정 없음 |
| 13년 이상 | 22/23 | 보정 없음 |

### 7.3 영역별 해석

**시공간/실행기능 (5점)**
- 0-2점: 심각한 장애
- 3점: 경도 장애
- 4-5점: 정상

**주의력 (6점)**
- 0-3점: 주의력 결함 시사
- 4-5점: 경계선
- 6점: 정상

**언어 (3점)**
- 0-1점: 언어기능 저하
- 2-3점: 정상 범위

**지연회상 (5점)**
- 0-2점: 기억력 장애 (알츠하이머형 치매 가능성)
- 3점: 경계선
- 4-5점: 정상

**지남력 (6점)**
- 0-4점: 지남력 장애 (중등도 이상 치매 시사)
- 5-6점: 정상

### 7.4 임상적 해석 원칙

1. **총점만으로 진단하지 않음**
   - MoCA-K는 선별도구이며 최종 진단은 종합평가 필요

2. **영역별 프로필 분석**
   - 특정 영역의 두드러진 저하는 특정 병리 시사
   - 예: 지연회상 저하 + 정상 지남력 = MCI 가능성

3. **위양성/위음성 고려**
   - 우울증, 불안, 교육 수준이 점수에 영향
   - 청력/시력 장애 배제 필요

4. **추적평가**
   - 6개월-1년 간격 재평가로 변화 추이 관찰
   - 4점 이상 하락 시 임상적으로 유의미한 악화

### 7.5 다른 검사와의 비교

| 검사 | MoCA-K | K-MMSE |
|-----|---------|---------|
| 난이도 | 상대적으로 어려움 | 쉬움 |
| MCI 민감도 | 높음 (89%) | 낮음 (18%) |
| 실행기능 평가 | 강함 | 약함 |
| 천장효과 | 적음 | 많음 |
| 교육 편향 | 중간 | 높음 |
