# PSS-10 (Perceived Stress Scale-10)
# 지각된 스트레스 척도

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **척도명** | PSS-10 (Perceived Stress Scale-10) |
| **한국어명** | 지각된 스트레스 척도 |
| **카테고리** | 스트레스 측정 |
| **문항 수** | 10문항 |
| **응답 시간** | 약 3-5분 |
| **대상** | 청소년 및 성인 |
| **평가 기간** | 지난 1개월 |

### 요인 구조
| 요인 | 문항 번호 | 설명 |
|------|-----------|------|
| 지각된 무력감 (Perceived Helplessness) | 1, 2, 3, 6, 9, 10 | 통제 불능 느낌 |
| 자기효능감 결여 (Lack of Self-efficacy) | 4, 5, 7, 8 | 문제 해결 능력 부재 |

## 5. 전체 문항

### 지시문
> 이 척도는 지난 한 달 동안 어떤 감정과 생각이 들었는지 물어보는 것입니다. 각 질문에 대해 당신이 얼마나 자주 그렇게 느꼈는지 표시해 주십시오.

### 문항 목록

| 번호 | 문항 (한국어) | 문항 (영어) | 역채점 |
|------|--------------|-------------|--------|
| 1 | 예상치 못했던 일 때문에 당황했던 적이 얼마나 있었습니까? | In the last month, how often have you been upset because of something that happened unexpectedly? | |
| 2 | 인생에서 중요한 일들을 조절할 수 없다는 느낌을 얼마나 경험하였습니까? | In the last month, how often have you felt that you were unable to control the important things in your life? | |
| 3 | 신경이 예민해지고 스트레스를 받고 있다는 느낌을 얼마나 경험하였습니까? | In the last month, how often have you felt nervous and "stressed"? | |
| 4 | **당신의 개인적 문제들을 다루는데 있어서 얼마나 자주 자신감을 느끼셨습니까?** | In the last month, how often have you felt confident about your ability to handle your personal problems? | ✓ |
| 5 | **일상의 일들이 당신의 생각대로 진행되고 있다는 느낌을 얼마나 경험하였습니까?** | In the last month, how often have you felt that things were going your way? | ✓ |
| 6 | 당신이 꼭 해야 하는 일을 처리할 수 없다고 생각한 적이 얼마나 있었습니까? | In the last month, how often have you found that you could not cope with all the things that you had to do? | |
| 7 | **일상생활의 짜증을 얼마나 자주 잘 다스릴 수 있었습니까?** | In the last month, how often have you been able to control irritations in your life? | ✓ |
| 8 | **최상의 컨디션이라고 얼마나 자주 느끼셨습니까?** | In the last month, how often have you felt that you were on top of things? | ✓ |
| 9 | 당신이 통제할 수 없는 일 때문에 화가 난 경험이 얼마나 있었습니까? | In the last month, how often have you been angered because of things that were outside of your control? | |
| 10 | 어려운 일들이 너무 많이 쌓여서 극복하지 못할 것 같은 느낌을 얼마나 자주 경험하셨습니까? | In the last month, how often have you felt difficulties were piling up so high that you could not overcome them? | |

> **역채점 문항 (긍정 문항)**: 4, 5, 7, 8번

## 6. 응답 척도

| 점수 | 한국어 | 영어 |
|------|--------|------|
| 0 | 전혀 없었다 | Never |
| 1 | 거의 없었다 | Almost never |
| 2 | 때때로 있었다 | Sometimes |
| 3 | 자주 있었다 | Fairly often |
| 4 | 매우 자주 있었다 | Very often |

## 7. 채점 방식

### 역채점 문항
**문항 4, 5, 7, 8번** 역채점:
```
역채점: 0→4, 1→3, 2→2, 3→1, 4→0
```

### 채점 공식
```
총점 = (Q1 + Q2 + Q3 + Q6 + Q9 + Q10) + [(4-Q4) + (4-Q5) + (4-Q7) + (4-Q8)]
```

### 하위척도 점수
```
지각된 무력감 = Q1 + Q2 + Q3 + Q6 + Q9 + Q10 (범위: 0-24)
자기효능감 결여 = (4-Q4) + (4-Q5) + (4-Q7) + (4-Q8) (범위: 0-16)
```

### 기본 정보
- **총점 범위**: 0-40점
- **고득점**: 높은 스트레스 지각
- **저득점**: 낮은 스트레스 지각

## 8. 해석 기준

| 총점 | 심각도 | 해석 |
|------|--------|------|
| 0-13 | 낮음 | 낮은 스트레스 수준 |
| 14-26 | 중등도 | 중간 정도의 스트레스 |
| 27-40 | 높음 | 높은 스트레스, 개입 고려 |

### 참고 규준 (미국 일반 인구)

| 30-44세 | 13.0 | 6.2 |
| 45-54세 | 12.6 | 6.1 |
| 55-64세 | 11.9 | 6.0 |
| 65세 이상 | 12.0 | 6.3 |
