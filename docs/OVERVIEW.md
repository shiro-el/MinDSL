# MinDSL - 심리 평가 도구를 위한 도메인 특화 언어

## 개요

MinDSL은 자기보고식 심리 평가 도구(설문지)를 정의하고 실행하기 위한 도메인 특화 언어(DSL)입니다.

### 설계 철학

> **"간단한 건 간단하게, 복잡한 건 가능하게"**

- **Progressive Disclosure**: 90%의 간단한 척도는 10-20줄로, 10%의 복잡한 척도는 전체 표현력으로
- **튜링 완전**: 어떤 채점 로직이든, 어떤 조건 분기든 표현 가능
- **듀얼 타겟**: 하나의 DSL 파일로 Next.js 웹앱과 Python 분석 환경 모두 지원

### 주요 특징

| 특징 | 설명 |
|------|------|
| **YAML 기반** | 배우기 쉽고 읽기 쉬운 선언적 문법 |
| **Python-like 표현식** | 익숙한 `and`, `or`, `==` 연산자 |
| **계층적 복잡성** | 단순 합산부터 IRT까지 5단계 복잡도 지원 |
| **다국어 내장** | 파일 내 다국어 텍스트 지원 |
| **자동 역채점** | `reverse: true` 플래그로 자동 처리 |
| **하위척도** | 문항 그룹핑 및 개별 채점 |
| **조건 분기** | Skip logic, 조건부 문항, 적응형 검사 |

## 타겟 사용자

1. **연구자**: 새로운 설문지를 쉽게 정의하고 배포
2. **개발자**: 설문 앱을 빠르게 구축
3. **데이터 분석가**: 기존 척도들을 통합 관리하고 자동 채점

## 지원 환경

- **웹**: Next.js 기반 설문 인터페이스
- **분석**: Python 런타임으로 데이터 처리 및 채점
