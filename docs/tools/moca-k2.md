# MoCA-K2 (몬트리올 인지평가 한국판 2판)

## 1. 기본 정보

**정식 명칭**: Montreal Cognitive Assessment - Korean Version 2 (MoCA-K2)
**목적**: 경도인지장애(MCI) 및 치매 선별을 위한 개정 인지기능 평가 도구
**소요 시간**: 약 10-15분
**대상**: 만 18세 이상 성인
**총점**: 30점 만점
**선별 기준점**: 연령 및 교육 수준에 따라 차등 적용

## 2. 원저자 및 출처

**원도구 개발**: Ziad Nasreddine, MD (MoCA 원저자)
**MoCA-K2 개발**: 송미경(Mi Kyoung Song) 연구팀
**개발 기관**: 충남대학교병원 정신건강의학과
**발표 연도**: 2022년

**주요 논문**:
- Song, M. K., et al. (2022). "Validation of the Korean Version of the Montreal Cognitive Assessment (MoCA-K) as a Screening Test for Both Dementia and Mild Cognitive Impairment in the Korean Elderly Population." *Psychiatry Investigation*, 19(6), 446-455.

**MoCA-K와의 차이점**:
- 한국 노인 인구에 최적화된 규준 제공
- 연령 및 교육 수준 보정 체계 개선
- 문화적 편향 최소화를 위한 문항 조정

## 5. 평가 영역 및 구조

MoCA-K2는 MoCA-K의 기본 구조를 유지하면서 문화적 적합성을 개선했습니다.

### 5.1 시공간/실행기능 (Visuospatial/Executive) - 5점

**1. 교대연결선로검사** (Alternating Trail Making) - 1점
- 숫자와 한글 자음을 교대로 연결
- 순서: 1-ㄱ-2-ㄴ-3-ㄷ-4-ㄹ-5
- 시간 제한: 없음 (정확도만 평가)
- 채점: 오류 없이 완료 시 1점

**2. 입방체 모사** (Cube Copy) - 1점
- 3차원 입방체 그림 따라 그리기
- 요구사항:
  - 모든 선이 그려져 있어야 함
  - 3차원 투시도법 정확해야 함
  - 선이 교차하는 지점이 정확해야 함
- 채점: 모든 조건 충족 시 1점

**3. 시계그리기** (Clock Drawing) - 3점
- **윤곽** (1점): 완전한 원 그리기
- **숫자** (1점): 1-12까지 숫자가 정확한 위치에
- **바늘** (1점): 11시 10분 정확히 표시
  - 시침: 11과 12 사이
  - 분침: 2를 가리킴

### 5.2 이름대기 (Naming) - 3점

- **동물 그림 이름대기**: 3개 동물 이름 말하기
- 문항 (한국 문화 적합성 고려):
  - 사자 (1점)
  - 코뿔소 (1점)
  - 낙타 (1점)
- 채점 기준:
  - 자발적 이름대기만 점수 부여
  - 단서 제공 시 0점
  - 지역 방언 허용

### 5.3 주의력 (Attention) - 6점

**1. 숫자 바로 따라하기** (Forward Digit Span) - 1점
- 숫자 5개: 2-1-8-5-4
- 1회 시행, 정확히 반복 시 1점

**2. 숫자 거꾸로 따라하기** (Backward Digit Span) - 1점
- 숫자 3개: 7-4-2
- 역순으로 정확히 반복 (2-4-7) 시 1점

**3. 주의집중** (Vigilance) - 1점
- 글자 읽기 과제: 검사자가 글자 목록 읽음
- 목표: '가'를 들을 때마다 손 두드리기
- 채점: 오류 0-1개 시 1점

**4. 계산** (Serial 7s) - 3점
- 100에서 7씩 연속 빼기
- 정답: 93, 86, 79, 72, 65
- 채점: 각 정답마다 1점 (최대 3점)
- 자기 교정 허용

### 5.4 언어 (Language) - 3점

**1. 문장 따라하기** (Sentence Repetition) - 2점

**문장 1** (1점):
- "나는 내가 언제나 다른 사람으로부터 도움을 받을 수 있다는 것을 알고 있다"
- 정확한 반복만 점수 부여

**문장 2** (1점):
- "고양이가 언제나 자기 영역 안으로 숨는다"
- 단어 추가/삭제/대체 불가

**2. 유창성** (Phonemic Fluency) - 1점
- 제한 시간: 60초
- 과제: '기' 또는 '시'로 시작하는 단어 말하기
- 기준: 11개 이상 시 1점
- 제외: 고유명사, 변형어, 반복

### 5.5 추상적 사고 (Abstraction) - 2점

**단어 쌍의 유사성 찾기**:

**1번 문항** (1점):
- 단어 쌍: "기차-자전거"
- 정답 예시: 탈것, 교통수단, 이동수단
- 오답 예시: 바퀴가 있다, 빠르다 (구체적 속성)

**2번 문항** (1점):
- 단어 쌍: "자-시계"
- 정답 예시: 재는 것, 측정도구
- 오답 예시: 숫자가 있다 (구체적 속성)

### 5.6 지연회상 (Delayed Recall) - 5점

**과제**: 검사 초반에 학습한 5개 단어 회상
- **학습 단계**:
  - 2회 반복 학습
  - 즉각 회상 실시 (점수 미포함)
- **회상 단계**: 약 5분 후 자유 회상
- **단어 목록** (버전 1):
  - 얼굴 (1점)
  - 벨벳 (1점)
  - 교회 (1점)
  - 데이지 (1점)
  - 빨강 (1점)

**채점**:
- 자유 회상만 점수 부여
- 범주 단서 제공하여 회상 시도 (참고용, 점수 없음)
- 재인 과제 시행 (참고용, 점수 없음)

### 5.7 지남력 (Orientation) - 6점

**시간 지남력** (4점):
- 날짜 (1점)
- 월 (1점)
- 연도 (1점)
- 요일 (1점)

**장소 지남력** (2점):
- 장소 (1점): 건물명 또는 기관명
- 도시 (1점): 현재 위치한 도시명

## 6. 채점 방식

### 6.1 원점수 계산

```
원점수 = 각 영역 점수의 합
       = 시공간/실행(5) + 이름대기(3) + 주의력(6) +
         언어(3) + 추상(2) + 지연회상(5) + 지남력(6)
       = 0-30점
```

### 6.2 보정 점수 계산 (MoCA-K2의 핵심 개선사항)

**연령 보정**:
- 연령 보정 없음 (규준 표에서 연령별로 구분)

**교육 보정**:
| 교육 연수 | 보정 점수 |
|---------|----------|
| 0년 (무학) | +2점 |
| 1-6년 | +1점 |
| 7-12년 | 0점 |
| 13년 이상 | 0점 |

**최종 점수 = 원점수 + 교육 보정 점수** (최대 30점)

### 6.3 채점 원칙

1. **즉각 채점**: 검사 중 즉시 채점 기록
2. **엄격한 기준**: 부분 점수 없음 (all-or-nothing)
3. **객관성 유지**: 모호한 경우 0점 처리
4. **표준화 시행**: 매뉴얼 지침 엄격히 준수

### 7.2 영역별 해석 기준

**시공간/실행기능** (5점)
- 0-2점: 실행기능 장애 시사 (혈관성 인지장애, 전두측두엽 치매 가능성)
- 3점: 경계선
- 4-5점: 정상 범위

**이름대기** (3점)
- 0-1점: 언어 능력 저하 (의미성 치매, 알츠하이머병 가능성)
- 2점: 경계선
- 3점: 정상

**주의력** (6점)
- 0-3점: 주의력 결함 (섬망, ADHD, 혈관성 인지장애 가능성)
- 4-5점: 경계선
- 6점: 정상

**언어** (3점)
- 0-1점: 언어기능 저하
- 2-3점: 정상 범위

**추상적 사고** (2점)
- 0점: 추상적 사고 장애 (전두엽 기능 저하)
- 1점: 경계선
- 2점: 정상

**지연회상** (5점)
- 0-2점: 기억력 장애 (알츠하이머형 치매 강력 시사)
- 3점: 경계선 (MCI 가능성)
- 4-5점: 정상

**지남력** (6점)
- 0-4점: 지남력 장애 (중등도-중증 치매 시사)
- 5점: 경계선
- 6점: 정상

### 7.3 프로필 패턴 해석

**알츠하이머형 치매 패턴**:
- 지연회상 ↓↓↓ (0-2점)
- 지남력 ↓↓ (시간 > 장소)
- 이름대기 ↓
- 다른 영역 상대적 보존

**혈관성 인지장애 패턴**:
- 실행기능 ↓↓↓ (교대연결선로, 시계)
- 주의력 ↓↓
- 지연회상 상대적 보존
- 변동성 큼

**전두측두엽 치매 패턴**:
- 실행기능 ↓↓↓
- 추상적 사고 ↓↓
- 언어 ↓↓ (유창성 특히)
- 기억 상대적 보존

**경도인지장애(MCI) 패턴**:
- 지연회상 ↓ (3점)
- 다른 영역 정상 또는 경계선
- 지남력 정상

### 7.4 종단 평가 해석

**변화의 임상적 유의성**:
- **정상 변동 범위**: ±2점
- **경도 악화**: 3-4점 하락 (6개월-1년)
- **중등도 악화**: 5-7점 하락
- **심각한 악화**: 8점 이상 하락

**치매 전환 예측**:
- MCI에서 1년 내 치매 전환률:
  - 총점 15-19: 약 40%
  - 총점 20-22: 약 20%
  - 지연회상 0-1점: 고위험군

### 7.5 다른 검사와의 관계

| 검사 | 상관계수 | 비고 |
|-----|---------|------|
| K-MMSE | r = .76 | MoCA가 더 어려움 |
| CERAD-K | r = .82 | 기억력 높은 상관 |
| CDR | r = -.71 | 역상관 |
| GDS | r = -.45 | 우울 영향 중간 |
