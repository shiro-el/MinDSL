# PANAS (Positive and Negative Affect Schedule)
# 정적 정서 및 부적 정서 척도

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **척도명** | PANAS (Positive and Negative Affect Schedule) |
| **한국어명** | 정적 정서 및 부적 정서 척도 |
| **카테고리** | 정서 측정 |
| **문항 수** | 20문항 (정적 정서 10 + 부적 정서 10) |
| **응답 시간** | 약 3-5분 |
| **대상** | 청소년 및 성인 |
| **평가 기간** | 시간 지시에 따라 다양 (현재, 오늘, 지난 며칠, 지난 주, 지난 몇 주, 지난 해, 일반적) |

## 4. 한국어판 정보 및 타당화 연구

### 주요 타당화 연구
- **이현희, 김은정, 이민규 (2003)**: 한국판 PANAS 최초 타당화
- **박홍석, 이정미 (2016)**: 수정된 한국판 PANAS (K-PANAS-R) 타당화

### 심리측정적 속성
| 지표 | 정적 정서 | 부적 정서 |
|------|----------|----------|
| Cronbach's α | 0.84-0.90 | 0.84-0.87 |
| 요인 구조 | 2요인 (PA, NA) |

### 한국판 특이사항
- 원척도의 '기민한(alert)' 문항이 한국판 2003년 연구에서 부적 정서 요인에 포함
- 2016년 수정판에서 원척도와 동일한 2요인 구조 확인

## 5. 전체 문항

### 지시문 (현재 순간 기준)
> 이 척도는 여러 가지 감정과 기분을 나타내는 단어들로 구성되어 있습니다. 각 문항을 읽고, 지금 이 순간 당신이 느끼는 정도를 1-5점 척도에서 표시해 주십시오.

## 6. 응답 척도

| 점수 | 한국어 | 영어 |
|------|--------|------|
| 1 | 전혀 그렇지 않다 | Very slightly or not at all |
| 2 | 약간 | A little |
| 3 | 보통 | Moderately |
| 4 | 상당히 | Quite a bit |
| 5 | 매우 많이 | Extremely |

## 7. 채점 방식

### 하위척도별 채점
```
정적 정서 (PA) = Q1 + Q3 + Q5 + Q9 + Q10 + Q12 + Q14 + Q16 + Q17 + Q19
부적 정서 (NA) = Q2 + Q4 + Q6 + Q7 + Q8 + Q11 + Q13 + Q15 + Q18 + Q20
```

### 기본 정보
- **정적 정서 범위**: 10-50점
- **부적 정서 범위**: 10-50점
- **역채점 문항**: 없음

### 점수 해석
- **정적 정서**: 높을수록 활력, 열정, 주의력 높음
- **부적 정서**: 높을수록 부정적 감정 경험 많음
- PA와 NA는 상대적으로 독립적인 차원

## 8. 해석 기준

### 일반적 해석
| 하위척도 | 점수 범위 | 해석 |
|----------|-----------|------|
| PA 10-20 | 낮음 | 낮은 긍정 정서, 무기력 |
| PA 21-35 | 보통 | 평균적 긍정 정서 |
| PA 36-50 | 높음 | 높은 긍정 정서, 활력 |
| NA 10-20 | 낮음 | 낮은 부정 정서, 평온 |
| NA 21-35 | 보통 | 평균적 부정 정서 |
| NA 36-50 | 높음 | 높은 부정 정서, 고통 |

### 정서 균형
- **PA 높음 + NA 낮음**: 최적의 정서 상태
- **PA 낮음 + NA 높음**: 우울/불안 위험 증가
- **PA 높음 + NA 높음**: 정서적 변동성 높음
- **PA 낮음 + NA 낮음**: 정서적 둔감 가능성
