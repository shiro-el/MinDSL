# DASS-21 (Depression Anxiety Stress Scales-21)
# 우울 불안 스트레스 척도

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **척도명** | DASS-21 (Depression Anxiety Stress Scales-21) |
| **한국어명** | 우울 불안 스트레스 척도-21 |

### 지시문

### 하위척도별 문항

#### 우울 (Depression) - 7문항
| 문항번호 | 문항 |
|----------|------|
| 3 | 긍정적인 감정을 전혀 느끼지 못했다 |
| 5 | 무엇을 시작하기가 어려웠다 |
| 10 | 기대할 것이 아무것도 없다고 느꼈다 |
| 13 | 슬프고 우울했다 |
| 16 | 어떤 것에도 흥미를 느끼지 못했다 |
| 17 | 나 자신이 가치 없는 사람이라고 느꼈다 |
| 21 | 인생이 무의미하다고 느꼈다 |

#### 불안 (Anxiety) - 7문항
| 문항번호 | 문항 |
|----------|------|
| 2 | 입이 마르는 것을 느꼈다 |
| 4 | 호흡이 힘들었다 (운동 없이도 숨이 가빴다) |
| 7 | 손이 떨렸다 |
| 9 | 나를 우스꽝스럽게 만드는 상황이 걱정되었다 |
| 15 | 공황 상태에 빠질 것 같았다 |
| 19 | 심장 박동을 느꼈다 (운동 없이도) |
| 20 | 이유 없이 두려웠다 |

#### 스트레스 (Stress) - 7문항
| 문항번호 | 문항 |
|----------|------|
| 1 | 사소한 일에 화가 났다 |
| 6 | 과민 반응하는 경향이 있었다 |
| 8 | 신경이 많이 예민해져 있었다 |
| 11 | 초조하다는 것을 느꼈다 |
| 12 | 긴장을 풀기가 어려웠다 |
| 14 | 방해받으면 참을 수 없었다 |
| 18 | 쉽게 짜증이 났다 |

## 6. 응답 척도

| 점수 | 한국어 | 영어 |
|------|--------|------|
| 0 | 전혀 해당되지 않았다 | Did not apply to me at all |
| 1 | 약간 해당되었다 | Applied to me to some degree |
| 2 | 상당히 해당되었다 | Applied to me to a considerable degree |
| 3 | 매우 많이 해당되었다 | Applied to me very much |

## 7. 채점 방식

### 하위척도별 채점
각 하위척도 점수 = 해당 7문항 합산

### ⚠️ 중요: DASS-42 변환
DASS-21 점수를 **2배**하여 DASS-42 기준으로 해석:
```
변환 점수 = 원점수 × 2
```

| 하위척도 | 문항 번호 | 원점수 범위 | 변환 점수 범위 |
|----------|-----------|-------------|----------------|
| 우울 | 3, 5, 10, 13, 16, 17, 21 | 0-21 | 0-42 |
| 불안 | 2, 4, 7, 9, 15, 19, 20 | 0-21 | 0-42 |
| 스트레스 | 1, 6, 8, 11, 12, 14, 18 | 0-21 | 0-42 |

### 역채점 문항
**없음** - 모든 문항 정방향 채점
