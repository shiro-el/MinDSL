## 5. 척도 구조

### 상태불안 척도 (State Anxiety Scale, S-Anxiety)
- 문항 번호: 1-20
- 측정 내용: 특정 순간에 느끼는 일시적 불안 상태
- 지시문: "지금 이 순간 어떻게 느끼는지"

### 특성불안 척도 (Trait Anxiety Scale, T-Anxiety)
- 문항 번호: 21-40
- 측정 내용: 비교적 안정적인 불안 성향
- 지시문: "평소에 일반적으로 어떻게 느끼는지"

### 샘플 문항 (공개 예시)
| 유형 | 불안 존재 문항 | 불안 부재 문항 (역채점) |
|------|---------------|----------------------|
| 상태불안 | "나는 긴장되어 있다" | "나는 평온하다" |
| 특성불안 | "나는 걱정이 많다" | "나는 안정된 느낌이다" |

> **참고**: 전체 문항은 저작권으로 보호되어 공개할 수 없습니다. 공식 검사지 구매가 필요합니다.

## 6. 응답 척도

### 상태불안 척도 (S-Anxiety)
| 점수 | 한국어 | 영어 |
|------|--------|------|
| 1 | 전혀 그렇지 않다 | Not at all |
| 2 | 조금 그렇다 | Somewhat |
| 3 | 보통으로 그렇다 | Moderately so |
| 4 | 대단히 그렇다 | Very much so |

### 특성불안 척도 (T-Anxiety)
| 점수 | 한국어 | 영어 |
|------|--------|------|
| 1 | 거의 그렇지 않다 | Almost never |
| 2 | 가끔 그렇다 | Sometimes |
| 3 | 자주 그렇다 | Often |
| 4 | 거의 언제나 그렇다 | Almost always |

## 7. 채점 방식

### 역채점 문항

#### 상태불안 척도 (10문항 역채점)
```
역채점 문항: 1, 2, 5, 8, 10, 11, 15, 16, 19, 20
정채점 문항: 3, 4, 6, 7, 9, 12, 13, 14, 17, 18
```

#### 특성불안 척도 (9문항 역채점)
```
역채점 문항: 21, 23, 26, 27, 30, 33, 34, 36, 39
정채점 문항: 22, 24, 25, 28, 29, 31, 32, 35, 37, 38, 40
```

### 역채점 공식
```
역채점 점수 = 5 - 원점수
(1→4, 2→3, 3→2, 4→1)
```

### 총점 계산
```
상태불안 점수 = Σ(정채점 문항) + Σ(역채점 문항의 역채점 점수)
특성불안 점수 = Σ(정채점 문항) + Σ(역채점 문항의 역채점 점수)
```

### 기본 정보
- **각 척도 점수 범위**: 20-80점
- **고득점**: 높은 불안
- **저득점**: 낮은 불안

## 8. 해석 기준

| 점수 범위 | 심각도 | 해석 |
|-----------|--------|------|
| 20-34 | 정상-낮음 | 낮은 불안 수준 |
| 35-54 | 정상 | 보통 수준의 불안 |
| 55-64 | 경미-중등도 | 임상적 주의 필요 |
| 65-80 | 중등도-심함 | 전문 평가 권장 |

### 임상적 절단점
- **55점 이상**: 임상적으로 유의한 불안 증상
- **상태불안 vs 특성불안 해석**:
  - 상태불안 높음: 현재 상황적 스트레스
  - 특성불안 높음: 만성적 불안 경향성
