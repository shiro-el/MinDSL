# BIS-11 (Barratt Impulsiveness Scale, Version 11)
# 배럿 충동성 척도

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **척도명** | BIS-11 (Barratt Impulsiveness Scale, Version 11) |
| **한국어명** | 배럿 충동성 척도 / 한국판 Barratt 충동성 검사 |
| **카테고리** | 충동성 측정 |
| **문항 수** | 30문항 |
| **응답 시간** | 약 10분 |
| **대상** | 청소년 및 성인 |
| **평가 기간** | 일반적인 행동 패턴 (특성 측정) |

## 5. 전체 문항

### 지시문
> 사람들은 서로 다른 방식으로 생각하고 행동합니다. 이것은 당신이 어떻게 생각하고 행동하는지를 알아보기 위한 검사입니다. 각 문항을 읽고, 얼마나 자주 그렇게 하는지 표시해 주십시오.

## 6. 응답 척도

| 점수 | 한국어 | 영어 |

| 4 | 거의 항상 그렇다 | Almost Always/Always |

### 요인 구조

#### 2차 요인 (Second-Order Factors)
| 요인 | 문항 번호 | 문항 수 |
|------|-----------|---------|
| 주의 충동성 (Attentional) | 5, 6, 9*, 11, 20*, 24, 26, 28 | 8 |
| 운동 충동성 (Motor) | 2, 3, 4, 16, 17, 19, 21, 22, 23, 25, 30* | 11 |
| 비계획 충동성 (Nonplanning) | 1*, 7*, 8*, 10*, 12*, 13*, 14, 15*, 18, 27, 29* | 11 |

#### 1차 요인 (First-Order Factors)
| 1차 요인 | 2차 요인 | 문항 번호 |
|----------|----------|-----------|
| 주의력 (Attention) | 주의 | 5, 9*, 11, 20*, 28 |
| 인지 불안정 (Cognitive Instability) | 주의 | 6, 24, 26 |
| 운동 (Motor) | 운동 | 2, 3, 4, 17, 19, 22, 25 |
| 지속성 (Perseverance) | 운동 | 16, 21, 23, 30* |
| 자기 통제 (Self-Control) | 비계획 | 1*, 7*, 8*, 12*, 13*, 14 |
| 인지 복잡성 (Cognitive Complexity) | 비계획 | 10*, 15*, 18, 27, 29* |

### 기본 정보
- **총점 범위**: 30-120점
- **고득점**: 높은 충동성
- **저득점**: 낮은 충동성

## 8. 해석 기준

| 총점 | 해석 |
|------|------|
| 30-50 | 낮은 충동성 (과통제 가능성) |
| 51-72 | 정상 범위 |
| 73-90 | 높은 충동성 |
| 91-120 | 매우 높은 충동성 (임상적 주의 필요) |

### 하위요인별 해석
- **주의 충동성**: 집중력 부족, 산만한 사고
- **운동 충동성**: 행동 억제 곤란, 즉흥적 행동
- **비계획 충동성**: 계획 부족, 미래지향 사고 부족
